# Later features

Ideas deliberately deferred until the Portfolio Copilot is proven useful:

- **Natural-language portfolio Q&A:** ask questions about holdings, exposure, and performance.
- **Daily market briefing:** summarize relevant market movement before the user reviews the portfolio.
- **Diversification suggestions:** identify concentration and suggest categories to research.
- **What-if analysis:** preview how a hypothetical trade changes allocation and risk.
- **Price-movement explanations:** explain unusual changes with relevant market context.

Add a feature only when it improves the live demo or a user asks for it.
